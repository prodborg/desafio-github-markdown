# Desafio Github Markdown
## Desafio proposto pela professora Aline Antunes no curso de Git e Github da DIO.

# README

O curso __Formação Github Certification__ apresentou diferentes funções e ferramentar para preparar seus alunos. Através dele foi possível entender a diferença entre Git e Github; como criar uma conta, repositórios e utilizar funções básicas do Github; ter acesso ao Git e conectar o Github através dele, além de aprender comandos necessários para as atividades de programadores que tem acesso a um mesmo diretório. A professora, __Alinne Antunes__, através de uma didática simples e detalhada, nos faz compreender a importância de cada comando e função em atividades básicas do cotidiano de um programador. A atividade em questão, nos faz exercitar muitas das habilidades compartilhadas por ela.
